## BauData — Encargado de Software
*2024 – 2026*

**BauData** es una plataforma SaaS de inteligencia de negocios orientada a la industria de la construcción en Chile, con seguimiento de obras, permisos de edificación y gestión de cartera.

Lideré el desarrollo Fullstack de la plataforma: construí la **API REST en NestJS**, desarrollé nuevas funcionalidades sobre la base de datos **MySQL** con consultas geo-espaciales, e implementé la **aplicación móvil en React Native** con mapas interactivos, visualización de proyectos, gestión de clientes y **notificaciones push** para alertas en tiempo real. Participé además en pipelines de **Web Scraping** con Selenium para ingesta automatizada de datos desde registros públicos.

La app móvil está disponible en:
[App Store](https://apps.apple.com/cl/app/baudata/id6748763426) • [Google Play](https://play.google.com/store/apps/details?id=com.baudata.ReactNative&hl=es_CL)

**Stack:** TypeScript, React Native, NestJS, MySQL, AWS (EC2, S3, RDS), Selenium, Git

## Proyecto Peuma — Facultad de Medicina, Universidad de Chile
*2026*

**Peuma** es una aplicación clínica crítica desarrollada para la detección de **delirio post-operatorio** en pacientes de quirófano. El proyecto, en colaboración con médicos e investigadores de la Facultad de Medicina, requirió los más altos estándares de seguridad y confiabilidad.

Desarrollé la aplicación en **Flutter** con integración **Bluetooth/IoT** para comunicación en tiempo real con dispositivos **Raspberry Pi** que capturan datos sensoriales del paciente durante el procedimiento. Diseñé una arquitectura con **encriptación de datos médicos sensibles**, administré el servidor privado Linux para visualización web, e implementé la lógica de negocio SaaS con expansión al ecosistema **iOS**.

**Stack:** Flutter (Dart), Bluetooth LE, Raspberry Pi, Linux Server, iOS/Android

## Neuronat — Facultad de Medicina, Universidad de Chile
*2025*

**Neuronat** es un videojuego clínico utilizado para la evaluación de **funciones ejecutivas** en pacientes. El proyecto consistí en una reingeniería completa: migrar el sistema desde **Unity** a **Godot** para mejorar la portabilidad, eliminar dependencias de licencias comerciales y facilitar el mantenimiento a largo plazo por el equipo médico.

Rediseñé la arquitectura de software desde cero, el sistema de almacenamiento y exportación de datos cognitivos, e implementé métricas de desempeño para análisis clínico posterior. El proyecto fue validado mediante **pruebas reales con usuarios** para garantizar fidelidad mecánica y consistencia funcional respecto al sistema original.

**Stack:** Godot (GDScript), diseño de arquitectura, testing con usuarios reales

## Detección de Baches — Proyecto Personal
*2025*

**Detección de Baches** es un sistema de visión computacional end-to-end para la **identificación automática de baches** en vías urbanas a partir de video capturado desde vehículos en movimiento.

Desarrollé una aplicación web en **React** desplegada en producción con **Netlify**, una API en **FastAPI (Python)** para la gestión y procesamiento de video, y un modelo predictivo basado en **YOLOv8** entrenado para detectar y clasificar baches en tiempo real. El sistema recibe grabaciones de cámaras vehiculares, las procesa automáticamente y devuelve los resultados georreferenciados con anotaciones visuales. Puedes visitarla aquí: [baches.netlify.app](https://baches.netlify.app/)

**Stack:** React, FastAPI, Python, YOLOv8, Netlify
